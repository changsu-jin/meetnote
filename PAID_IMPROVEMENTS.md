# MeetNote 유료 개선 방향

> 현재 MeetNote는 "추가 비용 0원" 원칙 하에 모든 처리를 로컬에서 수행합니다.
> 이 문서는 비용을 투자하면 달성 가능한 품질/성능 개선 방향을 정리합니다.

## 1. 전사 품질 (STT)

### 1-1. OpenAI Whisper API (클라우드)
- **예상 비용**: $0.006/분 (60분 회의 = ~$0.36)
- **개선 효과**: 고유명사 정확도 대폭 향상, hallucination 거의 없음
- **구현**: `transcriber.py`에 OpenAI API 백엔드 추가 (30줄 내외)
- **장점**: 로컬 GPU 불필요, 모델 다운로드 불필요
- **단점**: 인터넷 필수, 데이터가 OpenAI 서버로 전송됨

### 1-2. Deepgram / AssemblyAI API
- **예상 비용**: $0.0043~0.0065/분
- **개선 효과**: 한국어 전용 모델, 화자구분 내장 (diarizer 불필요)
- **구현**: 전사+화자구분을 단일 API 호출로 대체
- **장점**: 화자구분 정확도도 동시 개선, 실시간 스트리밍 지원
- **단점**: 프라이버시 포기

### 1-3. 커스텀 Whisper Fine-tuning
- **예상 비용**: GPU 서버 $50~200 (1회), 이후 로컬 추론 무료
- **개선 효과**: 도메인 특화 용어 정확도 극대화 (회사 프로젝트명, 인명 등)
- **구현**: HuggingFace 기반 fine-tuning → MLX 변환 → 기존 파이프라인 대체
- **장점**: 1회 비용 후 영구 로컬 사용, 프라이버시 유지
- **단점**: 학습 데이터 준비 필요 (50~100시간 오디오 + 전사본)

## 2. 화자 구분 (Diarization)

### 2-1. 화자구분 API (AssemblyAI / Deepgram)
- **예상 비용**: 전사 비용에 포함
- **개선 효과**: 짧은 발언도 정확한 화자 분리, 3명+ 화자 안정적 감지
- **현재 한계**: pyannote 로컬은 짧은 발언(< 2초)의 화자 분리에 약함

### 2-2. pyannote Pro (유료 라이선스)
- **예상 비용**: 문의 필요 (연구용 무료, 상업용 유료)
- **개선 효과**: 최신 모델 + 기술 지원
- **장점**: 로컬 처리 유지

## 3. 요약 품질

### 3-1. Claude API 직접 호출 (Anthropic)
- **예상 비용**: ~$0.003/1K input tokens (60분 회의 ~$0.15)
- **개선 효과**: Claude CLI 대비 파라미터 제어 (temperature, system prompt)
- **구현**: `summarizer.py`에 Anthropic SDK 백엔드 추가
- **장점**: 비동기 호출, 스트리밍, 구조화된 출력 (JSON mode)

### 3-2. GPT-4o API (OpenAI)
- **예상 비용**: ~$0.005/1K input tokens
- **개선 효과**: Claude와 비슷한 수준, 일부 한국어 태스크에서 우위
- **장점**: function calling으로 구조화된 요약 생성

## 4. 실시간 전사

### 4-1. Deepgram Streaming API
- **예상 비용**: $0.0043/분
- **개선 효과**: 현재 30초 청크 → 진짜 실시간 (< 1초 지연)
- **구현**: WebSocket 스트리밍으로 전사, 로컬 청크 방식 대체
- **장점**: 화자구분도 실시간, 지연 극소화

## 6. 추천 로드맵

### 프라이버시 우선 (데이터 외부 전송 불가)
1. **Whisper fine-tuning** (1회 $100~200) → 전사 정확도 극대화
2. **Claude API로 요약만** ($3/월) → 요약은 이미 익명화된 텍스트이므로 민감도 낮음

### 품질 우선 (비용 허용)
1. **Deepgram** ($5/월) → 전사+화자구분 한번에 해결
2. **Claude API** ($3/월) → 요약/교정

### 균형 (하이브리드)
1. 전사/화자구분은 로컬 유지 (현재 방식)
2. **LLM 교정만 API로** (현재 Claude CLI → Claude API 전환, $1/월)
3. Fine-tuning은 데이터 축적 후 진행
